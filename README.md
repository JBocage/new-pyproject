# new-pyproject

### Install

```
pip install git+https://github.com/JBocage/new-pyproject
```

### Use

Get all options using

```
new-pyproject --help
```

Example uses
```
new-pyproject backtest -a jbocage --fastapi-backend -i
new-pyproject backtest -a jbocage -d "A new project !" --ml-project --cli-entrypoint project-cli --cli-type command
```
